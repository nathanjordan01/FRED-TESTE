# 🧠 Gramática de Codificação: Mangueiras Montadas Dynar

> Extraído de: codificacao-da-mangueira-montada.pdf

## 🏗️ Estrutura do Código
O código segue o padrão:
`[TIPO]-[BITOLA]*[TERMINAL 1][MEDIDA]*[TERMINAL 2][MEDIDA]*[COMPRIMENTO] [ÂNGULO] [PROTEÇÃO]`

### 1. Tipos de Mangueira (Exemplos)
- **R01**: 1 trama de aço
- **R02**: 2 tramas de aço
- **R12**: 4 espirais de aço
- **4SH**: 4 espirais (Alta Pressão)

### 2. Tabela de Bitolas (Dash Size)
- **4**: 1/4"
- **6**: 3/8"
- **8**: 1/2"
- **12**: 3/4"
- **16**: 1"

### 3. Siglas de Terminais (Padrão)
- **PP00 / P00**: DKO Reto
- **PP90 / P90**: DKO 90°
- **PJ00 / J00**: JIC Giratória Reto
- **PF00 / F00**: Flange Reto (3000 psi)
- **PFR00 / FR00**: Flange Reforçado (6000 psi)

### 4. Proteções
- **1M/2M**: Manga anti-chama
- **MEP**: Mola plástica
- **DS**: Dispositivo de segurança

---
*Uso no Fred IA: O motor de busca deve quebrar inputs baseados nos separadores '-' e '*' para identificar componentes individuais.*
